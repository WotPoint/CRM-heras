🔐 Авторизация

Войдите для получения JWT-токена. Токен автоматически добавляется ко всем запросам.

POST /api/auth/login

Получить токен по email и паролю

Доступные аккаунты (пароль: 123456): manager@crm.ru · manager2@crm.ru · supervisor@crm.ru · admin@crm.ru

GET /api/auth/me

Текущий пользователь (требует токен)

👥 Пользователи

CRUD пользователей. Список — admin/supervisor. Создать/изменить/удалить — только admin.

GET/api/users

Список всех пользователей

GET/api/users/:id

Один пользователь по ID

POST/api/users

Создать нового пользователя (admin)


PATCH/api/users/:id

Обновить пользователя (admin)


DELETE/api/users/:id

Заблокировать пользователя — isActive = false (admin)

🏢 Клиенты

manager видит только своих клиентов; supervisor/admin — всех.

GET/api/clients

Список клиентов. Query: ?status=lead|active|regular|archived &managerId= &search=

GET/api/clients/:id
POST/api/clients

Создать клиента


PATCH/api/clients/:id

DELETE/api/clients/:id

Архивировать клиента (status → archived)

💼 Сделки

Смена статуса автоматически создаёт запись в истории. Удаление — только supervisor/admin.

GET/api/deals

Список сделок. Query: ?status= &managerId= &clientId=

GET/api/deals/:id
GET/api/deals/:id/history

История изменений статуса сделки

POST/api/deals
PATCH/api/deals/:id

Обновить сделку. Смена статуса → запись в истории


DELETE/api/deals/:id

Удалить сделку (supervisor/admin)

📋 Активности

Звонки, встречи, письма, заметки. Сортируются по дате (новые первые).

GET/api/activities

Query: ?type=call|email|meeting|note &clientId= &dealId=

GET/api/activities/:id
POST/api/activities
PATCH/api/activities/:id

DELETE/api/activities/:id

✅ Задачи

При смене статуса на done — автоматически ставится completedAt.

GET/api/tasks

Query: ?status=new|in_progress|done &priority=low|medium|high &assigneeId=

GET/api/tasks/:id
POST/api/tasks
PATCH/api/tasks/:id

Смена статуса на done → автоматически completedAt


DELETE/api/tasks/:id

📅 Календарь

События = задачи с дедлайном + активности, попадающие в диапазон дат.

GET/api/calendar?from=&to=

Список событий за период (задачи + активности)

📊 Отчёты

Доступны только supervisor и admin. Менеджер получит 403.

GET/api/reports/managers

Статистика по каждому менеджеру: клиенты, сделки, выручка, активности

GET/api/reports/funnel

Воронка продаж: количество и сумма сделок по каждому статусу

GET/api/reports/clients

Клиенты по статусам и источникам привлечения

GET/api/reports/activities

Активности по типам и менеджерам

💚 Health Check

Проверка доступности сервера.

GET/api/health

Не требует токена
